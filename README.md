# RFID-Scanner-Inventory-Management
An RFID Scanner to manage inventory

VS Code Extension:
PlatformIO

Library Dependencies:
|-- MFRC522 @ 1.4.12
    makuna/RTC@^2.5.0
|-- SPI @ 1.0
Components Used:

Arduino UNO R3

DS1302 RTC Module

USB Cable

MB-102 Breadboard

Jumper Wire

Resistance

Rc522 Module

White Card & Key Chain RFID

Data UID Lookup is stored in the Microcontroller's EEPROM


RC522 Connection (3.3v)
Digital:
PIN9  RST  - Reset (Yellow)

PIN10 SDA  - Slave Select (Green)

PIN11 MOSI - Master Out, Slave In (Blue)

PIN12 MISO - Master In, Slave Out (Orange)

PIN13 SCK  - Serial Clock (Gray)

RTC Module Connection (5V)
GND - Breadboard (5B -> 22B) (Black)

Digital:
PIN 5 CLK - Clock (Gray)

PIN 6 DAT - Data/IO (Orange)

PIN 7 RST - Reset (Yellow)
